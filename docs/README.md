# FastLLM Docs

The docs site is a Docusaurus app under `docs/`.

Content lives in:

- `docs/content/index.mdx` for the landing page
- `docs/content/guides/getting-started.mdx` for the first SDK example
- `docs/content/runtime/` for scheduler and local-model behavior
- `docs/content/reference/` for API reference notes

Released Rust API docs live on docs.rs.
